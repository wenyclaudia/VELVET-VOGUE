## ðŸ“ Deskripsi Project

**Velvet & Vogue** adalah purwarupa (prototype) website e-commerce yang dirancang untuk kategori **Fashion & Lifestyle**. Website ini dibangun menggunakan **HTML5, CSS3, dan JavaScript** dengan pendekatan *Single Page Application* (SPA) feel pada halaman utama, serta halaman detail produk yang terpisah.

Desain website mengusung tema *Modern Minimalist* dengan warna dominan **Deep Teal** dan tipografi elegan (*Playfair Display* & *Poppins*).

---

## âœ… Pemenuhan Kriteria Penilaian UAS

Project ini telah memenuhi seluruh kriteria penilaian sebagai berikut:

### 1. Komponen & Struktur Web (30%)
- **Menu & Dashboard:** Navbar responsif (Bootstrap) dan Hero Section yang menarik.
- **Info Section:** Bagian *About Us* dan *Our Story*.
- **Dynamic Gallery:** Galeri gambar dimuat secara dinamis menggunakan JavaScript (DOM Manipulation) dan Modal pop-up.
- **Form Input:** 
  - Form *Contact Us* (dengan validasi JavaScript).
  - Form *Newsletter Subscribe*.
- **API & Plugins:**
  - **Bootstrap 5:** Framework CSS untuk layout responsif.
  - **AOS (Animate On Scroll):** Library untuk animasi scroll yang halus.
  - **FontAwesome:** Ikon vektor untuk UI.

### 2. Desain & Kreativitas (30%)
- **Desain UI/UX:** Penggunaan palet warna konsisten, tipografi yang mudah dibaca, dan tata letak grid.
- **Struktur Halaman:** Terdiri dari lebih dari 5 section (`Home`, `Promo`, `Collection`, `Gallery`, `About`, `Contact`).
- **Halaman Detail:** Halaman `products.html` yang terhubung untuk menampilkan katalog lengkap dengan fitur filter kategori dan pencarian.

### 3. Integrasi & Hosting (30%)
- **Chatbot / Live Chat:** Integrasi tombol **Floating WhatsApp** yang mengarah langsung ke API WhatsApp (`wa.me`).
- **Google Analytics:** Implementasi script pelacakan trafik website (Global Site Tag `gtag.js`).
- **Hosting:** Dikonfigurasi untuk dapat di-hosting melalui **GitHub Pages**.
