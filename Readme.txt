DESKRIPSI PROJECT:
Website ini adalah platform e-commerce fashion modern yang dirancang 
menggunakan HTML5, CSS3, dan JavaScript. Website bersifat responsif 
(mobile-friendly) dan interaktif.

FITUR UTAMA (Memenuhi Kriteria Penilaian):
1. Navigasi & Dashboard: Single Page Application feel dengan Smooth Scroll.
2. Dynamic Gallery: Galeri gambar dimuat menggunakan JavaScript (DOM Manipulation).
3. Form Input: Contact Form dan Newsletter Subscribe dengan validasi dasar.
4. Plugins & API: 
   - Bootstrap 5 (Framework CSS)
   - AOS (Animate On Scroll Library)
   - FontAwesome (Icon Library)
   - Google Analytics (Tracking Script Implemented)
5. Integrasi Chat: Tombol Floating WhatsApp untuk komunikasi langsung.

----------------------------------------------------------------
CARA INSTALASI / MENJALANKAN:
1. Ekstrak file ZIP.
2. Buka file 'index.html' menggunakan browser (Chrome/Edge/Firefox).

CARA HOSTING (GITHUB PAGES):
1. Upload folder project ke Repository GitHub baru.
2. Masuk ke menu 'Settings' -> 'Pages'.
3. Pada bagian 'Branch', pilih 'main' atau 'master', lalu Save.
4. Website akan live di link yang diberikan GitHub.
